Day 5

Alice is a computer operator of the entrance gate at an old port where people rarely go. You out of nowhere, suddenly, want to have an evening off from work to go to that port. You go to that port and suddenly run into an unexpected problem.

The problem is: The gate is not opening because of an encryption error in the computer software that entrance gate is using. Alice feels sorry but is unable to resolve that problem, hence asks for help as you have introduced yourself as a software engineer to him. Now it is your duty to solve his problem and save your time off.

While you talk to Alice, he tells you that the encryption system works in such a way that I put my password in the password field and the gate open but as people rarely come to this port, the gate is locked since a while. The encryption system for the software of gate has outdated and needs to be renewed using the latest standards set by the owner of the gate’ software.

After a while of research, You and Alice come to this result that the txt file emailed to Alice is the password of this gate from now on (After the update). But, the txt file mailed to Alice is some random text which should be converted into a password using the particular methodology taught to Alice by the company at the time of recruitment.

That is as following:

  • The total non-empty lines of the txt means the first digits of the password. Like, if the txt contains 50 non empty lines, so the password’s first two digits will be 50
  • Whitespace doesn't matter once we get the max non empty lines.
  • All the vowels should be converted into numbers like A into 11, E into 22, I into 33, O into 44, U into 55. And all the consonants should remain as it is except ‘M’. M should be converted to 100.
  • After the conversion, the repeated occurrence of the same words/numbers should be deleted.
  • The password should not have any uppercase letter, they must be converted to lowercase.
  • Now the resultant should be set in alphabetical order and numbers should be placed in ascending order after the alphabets.
  • Remove any character other than number or alphabet.

Hurrah, we have got the password. The password’s format will be like NNAAAAAANN; where N represents a number and A represents alphabets. The first number will be obtained from rule one. The alphabets and trailing numbers from rule 4.
